# Product Data Exploration and Analysis using Microsoft Excel

This project is used to analyze and organize product-related data using Microsoft Excel, helping users understand product details, perform data analysis, and improve practical spreadsheet skills.

## Description

This project focuses on exploring and analyzing product-related information using Microsoft Excel. The purpose of this project is to understand how Excel can be effectively used to organize, manage, and analyze business-related product data in a structured manner.
The dataset used in this project contains important product details such as Product ID, Product Name, Brand Name, Category, Price, and Quantity. By using Excel formulas and spreadsheet techniques, meaningful insights can be obtained from the dataset.
This project also helps in developing practical knowledge of Excel by applying formulas, organizing tables, and understanding data patterns. Through this analysis, users can improve their spreadsheet management and problem-solving skills.

## Getting Started

### Dependencies

To use and run this project smoothly, ensure that the following requirements are available on your system:
* Operating System: Windows 10 or later
* Software: Microsoft Excel 2016 or above
* Knowledge Required: Basic understanding of Excel formulas and spreadsheet navigation
Having prior experience with Excel functions and table formatting will help in understanding the project more effectively.

### Installing

Follow the below instructions to install and access the project:
* Download the Excel workbook from the repository.
* Save the workbook file to your preferred location.
* Open the file using Microsoft Excel.
* If a protected view message appears, click Enable Editing.
* Verify that all sheets and formulas are functioning properly.
After completing these steps, the workbook will be ready for analysis.

### Executing program

Steps to Execute
To perform product data analysis successfully, follow these steps carefully:
* Open the Excel workbook.
* Navigate to the dataset worksheet.
* Observe the product details available in the table.
* Apply Excel formulas for calculations and analysis.
* Review product prices, categories, and quantity information.
* Analyze the outputs to understand product-related trends and data insights.
This process demonstrates how Excel tools can be used to simplify data analysis and improve accuracy.

### Example Formulas Used

* SUM Formula
The SUM formula is used to calculate the total of selected numerical values in a dataset.

Example:
```=SUM(D2:D34)```

* AVERAGE Formula
The AVERAGE formula helps in finding the mean value of selected data.

Example:
```=AVERAGE(D2:D34)```

* COUNTA Formula
The COUNTA formula is used to count non-empty cells in a selected range.

Example:
```=COUNTA(B2:B34)```

* MAX Formula
The MAX formula identifies the highest value present in the dataset.

Example:
```=MAX(D2:D34)```

* MIN Formula
The MIN formula is used to determine the smallest value in the selected range.

Example:
```=MIN(D2:D34)```
